# Redux Saga Demo

Redux Saga demo app.

Package includes:

- axios
- react
- react-redux
- react-router-dom
- react-scripts
- redux
- redux-saga
- sass
- eslint
- prettier
- redux-logger

## Getting Started

From the root folder...

```

npm install

npm start

```

Open `http://localhost:5439` in your favorite browser.
